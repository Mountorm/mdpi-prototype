交互说明 Interaction Description

操作流程 User Flow

  1. 用户点击稿件关联图标，输入 ID 选择稿件后保存。
  2. 用户关联稿件后，显示 Ethics Issue 关联选择器。
  3. 用户选择 Ethics Issue 并保存后，在右侧显示已关联的 Ethics Issue ID，并超链接跳转至 detail 页面。
  4. 点击铅笔图标按钮可以重新选择关联的 Ethics Issue。
  1. User clicks link icon on Manuscript ID, enters ID to select a manuscript and saves.
  2. After linking a manuscript, the Ethics Issue selector appears automatically.
  3. After selecting an Ethics Issue and saving, the linked Ethics Issue ID is displayed on the right with a hyperlink to the detail page.
  4. Click the pencil icon to re-select the linked Ethics Issue.

注意 Note

  • 如果取消关联了稿件 ID，Ethics Issue 自动解除关联。
  • 未关联 Ethics Issue 时不允许 Cancel，直接隐藏；编辑修改时才显示 Cancel。
  • 未选择Ethics Issue时，对勾按钮禁用不可点击。
  • If the manuscript ID is unlinked, the associated Ethics Issue is automatically removed.
  • When Ethics Issue is not linked, Cancel is hidden directly; Cancel only appears when editing.
  • When no Ethics Issue is selected, the check (confirm) button is disabled and cannot be clicked.
